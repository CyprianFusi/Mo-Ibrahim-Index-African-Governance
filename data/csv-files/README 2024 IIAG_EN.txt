2024 Ibrahim Index of African Governance (IIAG) data set
========================================================

This zip archive contains:

:: Raw data and raw data status for all variables
:: Score data for all series
:: Ranks for all series
:: Standard errors and confidence intervals at 80%, 85%, 90% and 95% for overall and category scores


For more information see: https://iiag.online/about.html


CSV FILE NOTES
==============

:: 2024 IIAG_Composite Scores.csv

Final composite scores for Overall Governance, four categories and sixteen sub-categories. All countries all years.
Scores are 0-100 where 100=best
Missing scores are expressed with "."

:: 2024 IIAG_Scores.csv

Scores for all IIAG series. Scores are 0-100 where 100=best
Missing scores are expressed with "."

PLEASE SEE THE 2024 IIAG_NAMING CONVENTIONS.CSV FILE FOR STRUCTURE AND SERIES IDs

:: 2024 IIAG_Ranks.csv

Ranks for all IIAG series. Ranks are 1-54 where 1=best
There are tied ranks when countries obtain the same score
Missing ranks exist when there is no data available for a country, expressed with "."

:: 2024 IIAG_Raw Data.csv

Raw data for all IIAG variables
Missing data points are expressed with "."

:: 2024 IIAG_Processed Data.csv

Trimmed raw data plus estimated datapoints

:: 2024 IIAG_Processed Data Type.csv

Status information for all IIAG variable data points

0= Data unavailable and cannot be estimated
1= Estimate
2= Raw data is used without processing (data available from source)
3= Trimmed data point
4= Trimmed estimated data point


:: 2024 IIAG_Confidence Intervals.csv

Standard errors and confidence intervals at 80%, 85%, 90% and 95% for overall and category scores
Confidence intervals are a range expressed in two parts, low and high.
